Sanitise Google Docs HTML

This tool cleans up the awful HTML that Google Docs spits out (preserving images, links, and footnotes), so you can use it somewhere else (blog post etc.).

Instructions

  1. Open your Google Doc
  2. In the Google Docs menu, select File → Download As → Web Page (HTML, zipped)
  3. Open the downloaded .zip file. This should extract an HTML file and a folder called 'images' (if your document contains them)
  4. Open the HTML file in a text editor (or open it in a web browser and click 'View → Developer → View Source')
  5. Copy the HTML into the 'input' field, set the options below, and click 'Sanitise'
  6. Copy your shiny, clean HTML from the 'output' field

The name of the folder you wish to put images associated with this document in. Change this if you want to rename the 'images' folder that accompanied the HTML file downloaded from Google Docs (e.g. mydoc_images), or if you want to store the images in a particular folder on your web server (e.g. 'sites/files/attachments/mydoc_images')
